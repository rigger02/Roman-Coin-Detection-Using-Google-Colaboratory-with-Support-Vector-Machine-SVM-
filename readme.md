# DETECTION COINS ROMAN

[![N|Solid](https://cldup.com/dTxpPi9lDf.thumb.png)](https://nodesource.com/products/nsolid)

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

This is a Roman coin detection project using digital image processing. The aim of this project is to distinguish a gold coin from a bronze coin in pictures using a computer algorithm.

## Features
1. Dataset Training and Testing
2. Pre-Process - Resampling 
3. Segmentation - Canny Edge Detection
4. Augmentation - Gaussian Blur
5. Feature Extraction - HSV (hue, saturation, value)
6. Algorithm - SVM (support vector machine)

## Note
This project is still under development and may be updated with additional features in the future. Make sure to check this repository periodically for the latest updates.
